# Everyday Notes

맥미니에서 실행하고 여러 기기에서 접속할 수 있는 웹 메모앱입니다.

## 지금 되는 기능

- 메모 작성, 수정, 삭제
- 제목과 내용 검색
- 브라우저 저장소에 자동 저장
- JSON 백업 내보내기/가져오기
- 한국어 음성 명령
  - `새 메모 오늘 회의 내용`
  - `요약`
- AI 요약 프록시 연결

## 맥미니에서 실행

앱 서버는 기본적으로 내부 포트 `1500`에서 실행됩니다.

```bash
export LLM_BASE_URL="http://115.137.55.154:90/v1"
export LLM_API_KEY="실제 키"
export LLM_MODEL="gemma4 e4b"
node ./server/summary-proxy.js
```

맥미니 안에서는 아래 주소로 확인합니다.

```text
http://localhost:1500
```

외부 접속은 공유기나 방화벽에서 아래처럼 포트 포워딩하면 됩니다.

```text
외부 포트 15 -> 맥미니 내부 IP:1500
```

이후 외부에서는 아래 형태로 접속합니다.

```text
http://외부주소:15
```

## GitHub 백업

GitHub에는 소스 백업용으로 올리는 것을 권장합니다. `.gitignore`에 `.env`가 들어가 있으니 API 키는 저장소에 올리지 마세요.

GitHub Pages로만 배포하면 서버 코드를 실행할 수 없어서 AI 요약 프록시는 따로 필요합니다. 맥미니에서 이 서버를 계속 켜두면 GitHub Pages 없이도 웹앱과 AI 요약을 같이 사용할 수 있습니다.

## AI 요약 설정

API 키를 브라우저 코드에 넣으면 다른 사람이 볼 수 있으니, 아래 구조로 동작합니다.

```text
브라우저 메모앱 -> 요약 서버 -> AI API
```

이 프로젝트의 Node.js 서버가 요약 서버 역할을 합니다.

앱은 현재 접속한 주소 기준으로 `/summarize`를 자동 호출합니다. 예를 들어 `http://외부주소:15`로 접속하면 요약 요청은 `http://외부주소:15/summarize`로 갑니다.

요약 서버는 내부적으로 아래 데이터를 받습니다.

```json
{
  "title": "회의 제목",
  "body": "회의 내용 전체"
}
```

그리고 아래처럼 응답하면 앱에 표시됩니다.

```json
{
  "summary": "회의 요약 내용"
}
```

## 기기 간 동기화에 대한 메모

현재 버전은 각 기기의 브라우저에 저장합니다. 모든 기기에서 같은 메모를 자동으로 동기화하려면 Firebase, Supabase, GitHub Gist, 또는 직접 만든 서버 저장소를 붙이면 됩니다.
